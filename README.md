# tsundere-coding
This project is a environment of interactive programming with tsundere prompt

## Example
```tsundere-coding
さあ、プログラミングを始めるわよ！
プログラムに何の命令を追加したいのよ？
1: 標準出力

5: 繰り返し
6: エラー処理
7: 関数定義
8: 値を返す
>>> 7
関数名は何にするのよ？
>>> x2
これって引数は取るの？
1: はい
2: いいえ
(デフォルトは2よ)
>>> 1
```
